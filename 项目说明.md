## 🚀 立即启动


## 💡 开发提示

### 添加新页面
1. 在 `src/views/` 创建新的 `.vue` 组件
2. 在 `src/router/index.js` 添加路由配置
3. 在需要的地方添加导航

### 修改样式
- 所有样式都在各自的 `.vue` 文件的 `<style scoped>` 中
- 使用 scoped 避免样式污染

### 调试技巧
- 使用 Vue DevTools 浏览器扩展
- 查看 Vite 的热更新日志
- 使用浏览器移动设备模式测试

## ⚠️ 注意事项
