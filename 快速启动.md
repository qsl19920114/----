# 🚀 智绘皮影 Vue3 版本 - 快速启动指南

## 📋 前置要求

- Node.js 16+ 
- npm 或 yarn 或 pnpm

## ⚡ 快速开始（3 步启动）

### 第 1 步：创建 public 目录并移动资源

```bash
# Windows CMD
mkdir public
xcopy assets public\assets\ /E /I

# PowerShell
mkdir public
Copy-Item -Path assets -Destination public\assets -Recurse

# Git Bash / Linux / macOS
mkdir -p public
cp -r assets public/
```

### 第 2 步：安装依赖

```bash
npm install
```

依赖列表：
- vue@^3.4.0
- vue-router@^4.2.0
- vite@^5.0.0
- @vitejs/plugin-vue@^5.0.0

### 第 3 步：启动开发服务器

```bash
npm run dev
```

浏览器会自动打开 `http://localhost:8080`

## 🐛 故障排除

### 问题 1: 图片无法显示
**原因**: 未将 assets 复制到 public 目录

**解决**:
```bash
mkdir public
cp -r assets public/
```

### 问题 2: 安装依赖失败
**原因**: npm 镜像源速度慢

**解决**: 使用国内镜像
```bash
# 使用淘宝镜像
npm install --registry=https://registry.npmmirror.com

# 或使用 cnpm
npm install -g cnpm --registry=https://registry.npmmirror.com
cnpm install
```

### 问题 3: 端口 8080 被占用
**解决**: 修改 `vite.config.js` 中的端口号
```javascript
server: {
  port: 3000,  // 改为其他端口
  // ...
}
```

### 问题 4: 路由跳转 404
**原因**: 开发服务器配置问题

**解决**: Vite 已自动配置，如果部署到服务器，需要配置 SPA 支持
